\# Week 2 - Linux Basics (Day1)

## 1. What is Linux?
Linux is an open-source operating system used in servers, cloud systems, DevOps pipelines, and containers.

## 2. Basic Linux Commands

-- 'pwd' -> Show current directory.

-- 'ls -la' -> List files with details
-- 'whoami' -> Show current user
-- 'hostname' -> Show system hostname
-- 'uname -a' -> Print system information

## 3. Linux File System Structure

-- '/home' -> User directiories
-- '/etc' -> Configuration files
-- '/var/log' -> System logs
-- '/usr/bin' -> Installed binaries
-- '/tmp' -> Temporary files
