# Git Notes - Week 1

# What is Git?

Git is a distributed version control system used to track changes in source code.

# What is repository?

A repository is a storage location where your code and its version history live.

# Working directory vs staging area

- Working directory: The files you actively edit on your computer.
- Staging area: The place where you prepare chanes before committing.

# Commit, branch, merge

- Commit: A snapshot of your changes.
- Branch: A separate line of development.
- Merge: Combining changes from one branch into another.

# Remote repository vs local repository

- Local rep: Your repository stored on your machine.
- Remote repo: Repository hosted on platforms like GitHub.
